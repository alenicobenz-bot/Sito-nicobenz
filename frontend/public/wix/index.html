Sezioni HTML per Wix.

Ogni sezione è un file HTML autonomo, pronto da incollare nel widget “Incorpora HTML” di Wix. Apri l'anteprima, scarica il codice e integralo manualmente nel tuo editor.

Come usarle in Wix

1. Dall'editor Wix, aggiungi un elemento Embed → Incorpora HTML nella posizione della sezione.

2. Clicca “Inserisci codice” e incolla il contenuto completo del file HTML (tag <html> incluso).

3. Imposta larghezza 100% e altezza generosa (es. 700–1200px) in base alla sezione.

4. Per adattamento automatico all'altezza, puoi usare l'app “Iframe Auto-Resize” dall'App Market di Wix.
